CSS Box Model

Property

Values

Description
box-sizing content-box Default value of box-sizing; Does not include padding and border to render the content's size
border-box It considers the size of the element as a whole, including padding and margin
overflow visible Default value, the content wil take extra space if it is bigger than its container
hidden Hides the content that overflows
scroll The overflowing content will be accessible via scroll bars within the original container
padding different units of measurement Defines the amount of space between the actual content and the border
margin different units of measurement Defines the amount of space between the border of the content and the rest of elements in the web
